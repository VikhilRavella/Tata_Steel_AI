OLLAMA SETUP GUIDE

This project uses local AI models through Ollama.

==================================================

STEP 1: Install Ollama

Download:

https://ollama.com/download

Install Ollama for your operating system.

==================================================

STEP 2: Download Required Models

Run the following commands:

ollama pull mistral:latest

ollama pull qwen2.5vl:latest

ollama pull qwen2.5-coder:7b

ollama pull BAAI/bge-small-en-v1.5

==================================================

STEP 3: Verify Models

Run:

ollama list

Expected Output:

mistral:latest

qwen2.5vl:latest

qwen2.5-coder:7b

BAAI/bge-small-en-v1.5

==================================================

STEP 4: Start Ollama

Run:

ollama serve

==================================================

==================================================

NOTE

Models only need to be downloaded once.

After installation, the application automatically selects the correct model based on the user's request.
