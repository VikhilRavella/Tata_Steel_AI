TATA STEEL INDUSTRIAL AI PLATFORM - ONE-CLICK INSTALLER GUIDE
==============================================================

This folder contains automated installer scripts designed to set up the entire platform from scratch with zero manual configuration. 
These scripts will automatically handle Python environments, dependency installation, Ollama configuration, and the downloading of all required AI models.

WINDOWS INSTRUCTIONS:
---------------------
1. Double-click "install.bat" (or run it from terminal).
   - This will automatically check python, create the virtual environment, install requirements, check/install Ollama, and download the 4 necessary open-source AI models.
2. Once complete, double-click "START_PLATFORM.bat".
   - This will launch both the backend API and frontend servers concurrently, and open your web browser automatically.

LINUX / MAC INSTRUCTIONS:
-------------------------
1. Open terminal and run:
   chmod +x install.sh
   ./install.sh
2. Once complete, run:
   chmod +x START_PLATFORM.sh
   ./START_PLATFORM.sh

TROUBLESHOOTING:
----------------
- If the installation fails, please check the newly generated `setup.log` file in this directory for detailed error messages.
- Note: Pulling the `qwen2.5-coder:7b` model may take considerable time depending on your internet connection (it is a large file). 
- Idempotency: The installer is fully idempotent. You can run it as many times as you like; it will simply skip steps (like model downloads) if it detects they are already complete.
