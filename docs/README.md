# Tata Steel Industrial AI Assistant

## Project Overview
An end-to-end AI platform for industrial maintenance, powering Field Engineers with vision diagnostics and document intelligence.

## Folder Structure
- `backend/`: FastAPI application, routers, services, database models, and local LLM integrations.
- `docs/`: Technical documentation and architecture mappings.

## Installation
Ensure you have Python and Ollama installed.
```bash
pip install -r requirements.txt
```

## Running Application
```bash
python start_app.py
```

## Manager Workflow
Upload Users, Equipment, SOP PDFs, and Inventory master data through the dashboard.

## Supervisor Workflow
Review and approve inventory requests and work orders generated by the AI on behalf of engineers.

## Engineer Workflow
Interact with the Engineering Agent via chat, log safety issues, and pull manuals on-demand.

## PDF Upload Flow
Managers upload manuals which are instantly chunked and vectorized via the BAAI embedding model into ChromaDB.

## Vision Analysis
Upload equipment images directly to the agent chat to automatically detect defects, risks, and recommendations via the multimodal Vision pipeline.

## Inventory Workflow
Agent automatically identifies needed parts from conversation or images and logs requests directly to the supervisor for approval.

## Demo Instructions
1. Run the local backend server.
2. Login as a Manager to initialize the database (Upload CSV/PDFs).
3. Login as an Engineer to start an AI diagnostic session.
4. Upload an image of damaged equipment to demonstrate the multimodal vision pipeline.
5. Query an SOP manual to demonstrate the local RAG pipeline.
